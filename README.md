# matrix.js
Matrix Digital Rain

- HTML 5 Canvas
- JavaScript
- CSS 3

Works well as a nice background to a website.
That's why the z-index is set to -1 on the canvas.
Opacity of the canvas could also be lowered to ~ 0.3 for a more subdued effect.
